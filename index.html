◆ 型について(その 2)

JavaScript のデータ型は大きく 5 つに分類されます。

Number 型(数値)：実数、浮動小数、10 進数、16 進数などの数値。

String 型(文字列)：テキストデータを表す型。表示の際は「' '」か「" "」で囲む

Boolean 型(真偽値)：データが正しいかを True(真)か False(偽)で表す。

Null 型(Null)：何のデータも含まれない状態。

undefined 型：何も定義されていない状態。

当面は、主に上 3 つを使用していきます。
JavaScript に限らず、型の理解は大切ですので、少しずつ慣れましょう。


console.log(typeof "text");
console.log(typeof 5);
console.log(typeof true);
console.log(typeof null);
console.log(typeof undefined);
演習 1-3
上記のコードをRunstantで実行し、結果を確認しましょう。
※ typeof は、データ型を判定する演算子です。

2. JavaScript 基礎文法(2)

◆ 演算子

数学で用いられる + や - などの記号の事です。
一般的な数式の使い方と似ていますが、異なる部分もあるため、
注意して学習を進めましょう。


・算術演算子

四則演算と剰余算数が含まれます。(剰余算：割り切れない余りを求める)

「+」：加算、「-」：減算、「/」：除算、「*」：乗算、「%」：剰余

console.log(1 + 2);
console.log(2 - 1);
console.log(4 / 2);
console.log(2 * 3);
console.log(5 % 2);

console.log("1" + "2");
演習 2-1
上記のコードを Runstantで実行し、結果を確認しましょう。

算術演算子を実行するとその計算結果が出力されますが、以下に示す演算子は、
式が正しい または 誤っている の結果が Boolean 型(真偽値)として、
True または False で出力されます。


・関係演算子

左辺と右辺間の値の大小を比較します。

[>]：大なり
> [<]：小なり
> [>=]：以上(大なりイコール)
> [<=]：以下(小なりイコール)


・等価演算子

左辺と右辺の式や値の一致・不一致を比較します。

[==]：等しい
> [!=]：等しくない


・論理演算子

2 つの命題の真偽値から、成立する条件を定める演算子。

[&&,and]：かつ、両方
> [||,or]：または、いずれか


console.log(1 < 2);
console.log(1 > 2);

console.log(1 == 1);
console.log(1 != 2);

console.log(1 > 2 && 2 < 3);
console.log(1 < 2 && 2 < 3);
console.log(1 > 2 || 2 < 3);

◆ 初期化と再代入

変数に値を格納する際、宣言と共に値を入れる事を「初期化」、
生成済みの変数に改めて値を入れる事を「再代入」といいます。

dollar = 105;
dollar = 110;
dollar = 107;
console.log(dollar);

◆ 変数の宣言

変数に値を格納する前に、変数の宣言が可能です。ES2015 以降の JavaScript では、
let と const が利用出来ます。let は再宣言が不可となり、const は再代入が不可となります。
const の方がより強力で、宣言後は「変数」ではなく「定数」となります。 

let only_one = 1;
let only_one = 2;
console.log(only_one);

const number = 1;
number = 2;
console.log(number);

演習 3-3
上記のコードを実行し、結果を確認しましょう。

また let と const 以外に、JavaScript では var という宣言が用いられていました。
そのため参考書や記事、フレームワークのマニュアルに var が用いられる事があります。
注意しましょう。


◆ インクリメントとデクリメント

数値に 1 ずつ追加させる処理をインクリメント、1 ずつ減らす処理をデクリメントといいます。
インクリメントが「変数++」デクリメントが「変数--」と記述します。

let i = 0;
i++;
console.log(i);

let j = 2;
j--;
console.log(j);

4. JavaScript 基礎文法(4)

◆if 文

プログラミングは原則、上から下へ順番に処理が流れますが、条件によって処理を分ける事ができます。
「もし〇〇なら A を実行する」といった形で、条件毎の処理を設定する記述法を条件分岐といいます。
その 1 つが if 文で、以下のように記述します。

if (条件式) {
  実行処理;
}

if 文の ( ) 内の条件が正しければ、(真偽値が True であれば)処理が実行されます。

if (2 > 1) {
  console.log("2は1より大きい");
  console.log(2 > 1);
}

演習 4-1
上記のコードを Runstantで実行し、結果を確認しましょう。



条件が複数ある場合は、else if を用いる事で記述が可能です。

if (条件式A) {
  実行処理A;
} else if (条件式B) {
  実行処理B;
} else if (条件式C) {
  実行処理C;
} else {
  実行処理D;
}

最後の else は 条件式 A ～ C までに当てはまらなかった全ての場合を含むため、条件式は不要です。
また、if 文内の条件式の記述は可読性を上げるため、Tab キー 1 回分の字下げを実行します。
